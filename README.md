## Introduction
This is the official implementation of the benchmark of Drone Fine-Grained Oriented Vehicle Detection(DFGOVD). We give the results of 12 OOD methods and release the models. Based on this dataset, we expect more researchers to join the study of fine-grained oriented vehicle detection methods.
